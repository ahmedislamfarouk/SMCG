# SMCG Visualizations

This folder contains generated visuals for project reporting and presentation.

## Files
- `smcg_weakness_baseline_vs_target.png`: Baseline vs target values for key identified weaknesses.
- `smcg_quality_metrics_snapshot.png`: Compact quality-metric snapshot chart for presentation use.

## Regeneration
Run from repository root:

```bash
python generate_visualizations.py --out-dir visualizations
```

## Notes
The values are report-oriented summary values aligned with documented weakness analysis and metric discussions.
